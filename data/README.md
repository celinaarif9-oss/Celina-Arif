# data/ — anonymised samples ONLY

⚠️ **Never commit real customer PII** (names, addresses, contacts, real order data) here or
anywhere in the repo. This folder holds **anonymised / synthetic samples only**, used to model
sheet columns, templates, and calculations.

- Anything genuinely private goes in `data/private/` (gitignored) or stays out of the repo.
- When in doubt, anonymise: replace names/addresses/contacts with placeholders.

See `../CLAUDE.md` → "Security & data".
